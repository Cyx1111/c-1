Windows Phone SDK 8.0

使用 Windows Phone SDK 8.0 为 Windows Phone 8 或 Windows Phone 7.5 生成应用程序。

Windows Phone SDK 8.0 可扩展 Visual Studio 2012 以便为您提供 为 Windows Phone 开发应用程序所需的工具，其中包括 Windows Phone 应用程序和库模板、Windows Phone 仿真程序及性能 分析工具。可以使用此 Visual Studio 扩展来编译和调试 Windows Phone 应用程序，然后将其部署到 Windows Phone 或 Windows Phone 仿真程序。

注意    出现 该页的项目中包含有关安装 Windows Phone SDK 8.0 的说明而不是该 SDK 本身。若要下载该 SDK，请单击 下载 Windows Phone SDK 8.0 按钮，或转到 [download page URL]。
