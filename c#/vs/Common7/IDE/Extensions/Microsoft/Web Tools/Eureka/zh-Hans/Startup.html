Page Inspector

Page Inspector 可让你在 Visual Studio 中对 Web 项目进行诊断和故障排除。若要开始使用，请打开一个项目或网站并单击“主页”按钮，或者右键单击一个文件并选择“在 Page Inspector 中查看”。需要深入了解的某些功能包括:

  • 检查模式

    单击“检查”按钮可轻松分解你的页面，并将元素映射回源中的确切位置。

  • CSS 映射

    在“样式”或“跟踪样式”选项卡中单击一个 CSS 规则，然后在项目的源文件中找到确切的位置。

  • 快速刷新浏览器

    按 Ctrl+Alt+Enter 可保存所有文件并刷新 Page Inspector 的浏览器，而无需退出编辑器。
