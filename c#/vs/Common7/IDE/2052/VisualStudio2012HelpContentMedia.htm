使用 Microsoft Help 查看器 2.0

·       MSDN 内容的产品交付 DVD 包含针对 Microsoft Help 查看器 2.0 的内容，该查看器随 Visual Studio 2012 一起安装。您无法使用 Microsoft Help 查看器 1.1 安装此内容。

·       您不能重新分发 Help 查看器 2.0，它只能与 Visual Studio 2012 中的某些产品一起安装。

·       在一些安装中，管理员可能已禁用管理内容选项卡。在这些情况下，您无法通过使用本文中的说明来安装帮助内容。您必须向管理员咨询。

·       要安装帮助内容，您必须具有本地计算机的管理权限。

使用 ISO 映像文件

许多产品都以 ISO-9660 映像文件的形式发布，这正是 CD 或 DVD 的表现形式，包括内容和逻辑格式。通常，用户将映像文件写入空白 CD-R 或 DVD-R，生成与原始磁盘相同的副本，包括文件名和卷标。您也可以打开 ISO 映像文件并将其内容复制到本地文件夹，就像对压缩的 .zip 文件那样。或者，您还可以以设备的形式虚拟安装和访问 ISO 文件。根据您的软件，您可能需要将文件扩展名从 .img 或 .udf 重命名为 .iso。

您用于写入 CD-R 和 DVD-R 的软件可能包括将映像复制到 CD或刻录映像的功能，您可以使用该功能从映像文件创建磁盘。有关详细信息，请参见所用软件的帮助。创建磁盘之后，请完成下一节中适当的一组步骤。

从磁盘安装文档

您可以采用以下方式之一从 MSDN 内容的产品交付 DVD 安装内容。

对于安装到本地计算机的个人：

1.  将 DVD 插入计算机的 DVD 驱动器中。

2.  在 Visual Studio 2012 的菜单栏上，依次选择帮助以及添加和移除帮助内容。

随即启动 Help 查看器 2.0，显示管理内容选项卡。

3.  在管理内容选项卡上，选择磁盘选项按钮，选择浏览()按钮，然后导航到 DVD 驱动器。

4.  选择 helpcontentsetup.msha 文件，然后选择打开按钮。

在可用文档下，会显示来自 DVD 的可用内容集的列表。

5.  选择想要安装到本地计算机上的内容集旁边的添加链接。

您指定的项将会出现在挂起的更改列中。可以通过选择项旁边的X将该项从挂起的更改列中删除。

6.  选择更新按钮。

对于企业环境：

产品交付 DVD 包含内容和一个 .msha 文件，您可以将这些内容复制到用户可以访问的网络位置。或者，您也可以设置客户端默认指向此位置，从而使每个客户端均可以管理其各自的本地内容。有关如何管理用户本地计算机上的帮助的更多信息，请参见 Help 查看器 2.0 联机管理员指南。
